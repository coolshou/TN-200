2.2 omp_get_max_threads – Maximum number of threads

Description:
Return the maximum number of threads used for parallel regions that do not use the clause num_threads.
C/C++:

Prototype: int omp_get_max_threads();

Fortran:

Interface: integer function omp_get_max_threads()

See also:
omp_set_num_threads, omp_set_dynamic
Reference:
OpenMP specifications v2.5, section 3.2.3.
