1.Please follow the below procedure to build a working f/w :
   1.1 tar zxvf tn200_GPL.tgz
   1.2 cd tn200_GPL
   1.3 source build_fw
   Note : a. make sure uRamdisk, uImage, image.cfs,  and default.tar.gz are resided within the merge folder, the default name of generated F/W is : TNA-200-firmware.
          b. upload the TNA-200-firmware through device GUI
